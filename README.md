# Тестовое задание для Гринатом
### Задание 1
Причин возникновения ошибки 500 может быть множетство. Тем не менее, вот несколько базовых шагов, которые я бы предпринял:
1. Проверить логи сервера.
2. Убедиться, что сервер настроен правильно и имеет все необходимые зависимости. 
3. Проверить результаты тестов.
4. Обратиться за помощью к хостинг-провайдеру.

### Задание 2
Проблема заключается в том, что замыкание в цикле ссылается на переменную step, а не на ее значение в цикле. 
В результате чего к моменту выполнения обработчиков значение step равно 4. 
Чтобы исправить это, мы можем передать step в качестве аргумента в лямбда-функции.
